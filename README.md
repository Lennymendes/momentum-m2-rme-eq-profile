# momentum-m2-rme-eq-profile

Community headphone EQ profile for the **Sennheiser Momentum M2 (wired)** based on measurements from InnerFidelity, designed for RME audio converters.

---

[![POSTER.png](https://i.postimg.cc/7Yv96fFQ/POSTER.png)](https://postimg.cc/23wh9884)



## Compatible devices (via RME Remote Control)

* ADI-2
* ADI-2 Pro
* ADI-2 Pro FS
* ADI-2 Pro AE
* ADI-2 Pro FS Black Edition
* ADI-2 Pro EX

## Installation / Import / Load (RME ADI Remote workflow)

1. Open the **RME Remote Control app**.
2. Go to the **EQ / Room EQ preset section**.
3. Select an empty preset slot.
4. Load the file: `Momentum M2`
5. Store / save the preset to the device.
6. Enjoy the corrected sound profile.


## Notes

* This profile is intended for use with **RME DSP EQ only**.
* It is based on community measurements and subjective tuning.
* Small adjustments may be needed depending on your headphones' pad wear or hearing preference.
