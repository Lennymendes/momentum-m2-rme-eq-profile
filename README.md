# momentum-m2-rme-eq-profile

Community headphone profile based on innerfidelity for the Sennheiser Momentum M2 WIRED on RME audio converters.

Compatible devices for this profile with REMOTE CONTROL by RME : 

-ADI-2, ADI-2 pro, ADI-2 PRO FS, -ADI-2 PRO AE, -ADI-2 PRO FS Black Edition, -ADI-2 PRO EX


Installation

1. Open TotalMix FX on the host computer.
2. Select the headphone output.
3. Open EQ.
4. Enter the filter settings listed below.
5. Save the configuration as a Snapshot or Workspace.
6. Access the profile remotely through TotalMix Remote.
